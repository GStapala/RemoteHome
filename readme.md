# Remote home

Mobile app made in Xamarin forms with Web api 2 backend.

<img src="http://i.imgur.com/gf35NVA.gif"/>


## Technologies used :
* Xamarin forms
* Prism - Mvvm 
* NGraphics - Animations
* Unity - IoC
* Web api 2
* log4net
*
